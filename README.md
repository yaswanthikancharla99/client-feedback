# 💬 Client Feedback & Analytics Portal

## 📌 Overview

The **Client Feedback & Analytics Portal** is a frontend web application developed using **HTML, CSS, and JavaScript**. This portal allows organizations to collect client feedback, store responses, and analyze ratings to understand overall service quality.

It helps in identifying areas for improvement and enhances customer satisfaction through feedback analysis.

---

## 🚀 Features

* Submit client feedback
* Select service category
* Provide rating (1–5)
* View feedback in a table
* Calculate total feedback count
* Calculate average client rating
* Simple and user-friendly interface
* Fully frontend (no backend required)

---

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript

---

## 📊 Functional Modules

### 1. Feedback Submission

Clients can enter:

* Client Name
* Service Type
* Rating
* Feedback Message

### 2. Feedback Management

* Displays feedback in table format
* Organizes client responses

### 3. Analytics

* Total number of feedback entries
* Average rating calculation
* Helps evaluate service performance
